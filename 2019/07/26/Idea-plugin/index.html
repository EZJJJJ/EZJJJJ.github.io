Idea插件推荐

1. Alibaba Java Coding Guidelines（阿里巴巴Java开发规约）

价格：

​ 免费

### 用途：

​ 用于检查代码是否规范。包括命名，方法调用，注释等等很多方面的检查。可以让代码更规范。

2. MyBatisCodeHelperPro

价格：

​ 29/年

用途：

​ 这款插件对于使用MyBatis的开发者来说可是个宝贝了。可以通过mapper接口直接找到对应的xml内的方法，免去了在一大堆XML文件中寻找的苦恼。当Idea成功连接数据库后，可以一键生成对应表的JavaBean，Xml，Mapper接口和Service。

​ 当然他的功能还不止这些，官方文档中还有更多的介绍。

3. GsonFormat

价格：

​ 免费

用途：

​ 根据JSON生成对应的JavaBean。

4.GenerateAllSetter

价格：

​ 免费

用途：

​ 一键调用一个对象的所有set方法。

​
